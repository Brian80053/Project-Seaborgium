# Project-Seaborgium
1-1 C언어 프로그래밍 시간 프로젝트로 하게 된 프로젝트 시보귬 - CLI 지뢰찾기 만들기 프로젝트 

## 프로젝트 소개
기존의 지뢰 찾기는 중독성이 있지만, 플레이 방식이 단조로워 얻을 수 있는 재미가 한정적이라는 점이 아쉬웠다. 따라서 기존의 지뢰 찾기를 포함해 CLI 내에서 작동하는 여러 종류의 지뢰 찾기를 만들고 싶었다.

## 주요 기능
지뢰 찾기를 기본 모드만 만들기에는 너무 단조로웠다. 이를 대신하기 위해 6개의 모드를 제작하였다.
### 기본 모드
설명할 필요가 없는 기본 모드이다. F/N (알파벳 X좌표) (알파벳 Y 좌표)를 통해 입력 할 수 있고, 플래그 모드와 일반 모드를 바꿀 수 있다. 플래그 모드로 빈칸에 깃발을 꽂을 수 있고, 일반 모드는 빈 칸을 팔 수 있다. 숫자는 주변 8칸의 지뢰 갯수를 뜻한다. 빈 칸을 다시 지정하면 주변 8칸에 올바른 위치에 플래그가 나올 시 주변을 밝혀주는 편의 기능이 있다.
### 랜덤 모드
기본 모드와 비슷하지만 세로 크기와 지뢰 개수는 랜덤이다. 지뢰 개수는 전체 수의 절반 미만으로 설정했다.
### 노플래그 모드
플래그를 사용할 수 없다. 플래그를 사용할 시 게임 오버되고, 이외의 것들은 기본 모드와 같다.
### 와드 모드
와드 모드는 플래그를 사용할 수 없는 노플래그 모드의 강화판이며, 화면이 아예 보이지 않는 대신 10개의 와드를 설치해 주변 3x3을 볼 수 있다. 매우 어렵다.
### 대전차 지뢰 모드
와드 모드와 더불어 가장 특이한 모드이다. 지뢰중 19개가 대전차 지뢰로 바뀐다. 대전차 지뢰는 일반 지뢰와 달리 클릭해야 살아남고 플래그를 꽂으면 게임 오버가 된다. 대전차 지뢰는 색깔로 구분할 수 있으며 주변 3x3칸 중 1~6개의 경우 각각 빨강→초록→노랑→파랑→마젠타→시안색으로 표기되며, 7개는 기울임, 8개는 볼드로 표기된다.
### 타임 어택 모드
10초 안에 행동을 하지 않는다면 바로 게임 오버가 된다. 이외의 것들은 기본 모드와 같다.

## 기술 스택
모든 코드는 C로 작성되었으며, 다른 사람들의 프로젝트와 다르게 Python이나 json 등의 다른 언어를 일체 사용하지 않았다.

## 실행 방법
exe 파일을 다운로드 받은 뒤 실행하면 된다. 알 수 없는 사용자는 무시하고 실행하면 된다. 아직은 임시 실행 방식이고, 추후에 git을 활용한 설치/실행 방식으로 바꿀 것을 고려 중이다.
